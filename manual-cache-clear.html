🧹 캐시 문제 해결 가이드

YouTube Pulse의 캐시 문제를 해결하는 단계별 방법입니다.

⚠️ 중요: 다음 단계를 순서대로 진행하세요.

1단계: 개발자 도구 열기

키보드에서 F12 키를 누르거나, 브라우저 메뉴에서 개발자 도구를 열어주세요.

2단계: 서비스 워커 해제

DevTools에서 Application 탭 → Service Workers → 해당 도메인에서 Unregister 클릭

3단계: 저장소 정리

DevTools에서 Application 탭 → Storage → Clear site data 클릭

4단계: 강제 새로고침

DevTools가 열린 상태에서 새로고침 버튼을 길게 클릭 → "Empty Cache and Hard Reload" 선택

5단계: 확인

페이지가 새로 로드되면 데이터 분류 관리 페이지에서 데이터가 정상 표시되는지 확인하세요.

메인 페이지로 이동 강제 새로고침
💡 팁: 만약 위 방법으로 해결되지 않으면, 브라우저를 완전히 종료하고 다시 실행해보세요.
