# YouTube Pulse 배포 가이드

## 다른 PC에서 실행하는 방법

### 방법 1: 개발 환경으로 실행 (권장)

1. **필요한 파일들:**
   - 전체 프로젝트 폴더 (`C:\youtubepulse`)
   - 또는 압축 파일로 전송

2. **설치 및 실행:**
   ```bash
   # 프로젝트 폴더로 이동
   cd C:\youtubepulse
   
   # 의존성 설치
   npm install
   
   # 앱 실행
   npm run electron:dev
   ```

3. **시스템 요구사항:**
   - Node.js (v16 이상)
   - npm 또는 yarn

### 방법 2: 웹 버전 실행

1. **dist 폴더 내용을 웹 서버에 업로드**
2. **브라우저에서 접속하여 사용**
